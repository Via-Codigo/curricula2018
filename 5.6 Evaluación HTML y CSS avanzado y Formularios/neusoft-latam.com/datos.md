## Tipografía

Estas son las reglas para la tipografía

```
body{
    font-family: sans-serif;
    font-size: 16px
}

h1{
    font-size: 120px
}

.modal-titulo{
    font-size: 32px;
}

.modal-texto{
    font-size: 18px;
}
```

El tamaño de la letra es 16px en todas partes excepto en el título de contáctenos en que es 120px;
